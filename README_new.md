## 常见问题

跟开源版的区别是什么？

1、鼠标控制跟地图差不多，适合大场景；  
2、支持 WebGPU 渲染；  
3、上传模型支持上传文件夹；  
4、代码编辑器改成 vscode 用的 monaco-editor 了；  
5、改为文件存储，不需要数据库；  
6、后端改成了 node.js 实现，只要 node.js 就可以编译前后端。
